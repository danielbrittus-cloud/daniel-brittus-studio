# Daniel Brittus Studio

Dark glassmorphism portfolio site for Daniel Brittus Studio, featuring playable HTML5 browser games.

## Featured Game

**Tower Build** is an arcade tower-building game where players stack houses, survive changing weather, climb past the atmosphere, and reach space.

Playable route:

```text
/games/tower-build/
```

## Local Preview

```powershell
node dev-server.cjs
```

Open:

```text
http://127.0.0.1:4177/
```

## Vercel

This is a static site. Import the repository into Vercel and deploy with the default settings.

No build command is required.
